# Paper 1: Diseño in silico de vector CRISPRa para resistencia en cacao

## Descripción
Diseño racional in silico de un vector de edición génica (CRISPRa dCas9-VP64)
para sobreexpresión de TcNPR1 en Theobroma cacao y conferir resistencia
a Moniliophthora roreri (frosty pod rot).

## Estructura del proyecto
- data/raw/       : datos crudos del SRA (solo lectura)
- data/genome/    : genomas de referencia
- data/processed/ : datos procesados
- results/        : figuras y tablas
- workflow/       : pipeline Snakemake y scripts
- envs/           : entornos Conda
